# 💼 Worker Contracts

Sistema desenvolvido em **Java** com o objetivo de aplicar conceitos fundamentais de **Programação Orientada a Objetos (POO)**, como herança, composição e manipulação de datas.

O projeto simula o cálculo de ganhos de um trabalhador com base em seu salário base e contratos adicionais por hora.

---
## 🎯 Objetivo

Demonstrar na prática conceitos importantes de orientação a objetos, incluindo:

- Associação entre entidades (Worker e HourContract)
- Uso de composição
- Cálculo de renda mensal
- Manipulação de datas com `LocalDate`

---
## 🏗️ Estrutura do Projeto
```text
Worker_Contracts/
│
├── application/
│   └── Program.java
│
├── entities/
│   ├── Worker.java
│   ├── Department.java
│   └── HourContract.java
│
└── enums/
    └── WorkerLevel.java
```
---
## 🚀 Funcionalidades
- Cadastro de trabalhador
- Associação com departamento
- Definição de nível do trabalhador (enum)
- Registro de contratos por hora
- Cálculo de renda mensal com base nos contratos
- Manipulação de datas (LocalDate)
- Uso de listas (ArrayList)
---

## 🧠 Conceitos Aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Composição
= Enumerações
- Coleções (List)
- Manipulação de datas (LocalDate)
- Entrada de dados com Scanner
- Organização em camadas simples
---

# ENGLISH 
---
# 💼 Worker Contracts

A system developed in **Java** focused on applying core **Object-Oriented Programming (OOP)** concepts such as inheritance, composition, and date handling.

The project simulates a worker's income calculation based on a base salary and additional hourly contracts.

---
## 🎯 Purpose

To demonstrate important OOP concepts, including:

- Entity relationships (Worker and HourContract)
- Composition
- Monthly income calculation
- Date handling with `LocalDate`
---

## 🏗️ Project Structure

```text
Worker_Contracts/
│
├── application/
│   └── Program.java
│
├── entities/
│   ├── Worker.java
│   ├── Department.java
│   └── HourContract.java
│
└── enums/
    └── WorkerLevel.java
```
---

## 🚀 Features
- Worker registration
- Department association
- Worker level definition (enum)
- Hourly contract registration
- Monthly income calculation
- Date manipulation (LocalDate)
- Use of collections (ArrayList)
---

## 🧠 Concepts Applied
- Object-Oriented Programming (OOP)
- Encapsulation
- Composition
- Enumerations
- Collections (List)
- Date handling (LocalDate)
